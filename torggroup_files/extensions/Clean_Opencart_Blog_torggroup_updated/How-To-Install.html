How to install

Please follow all steps carefully

Step 1: Upload the filesUpload the folders from the folder Files-To-Upload/ to the root of your store installation. The structure of the folders is the same as in opencart. No files will be overwritten.

Step 2: Import clean_blog.ocmod.xmlLog in to your admin panel, and go to Extensions > Extension Installer.

Click on browse, and select the file clean_blog.ocmod.xml. You should should get the following message:

Success: You have installed your extension!

Step 3: Refresh Modification CacheGo to Extensions > Modifications. Hit the blue button at top right that says "Refresh".

You should now see the Blog added to your admin menu

Step 4: Set PermissionsGo to System > Users > User Group. Edit the Administrator account, and make sure all blog areas are checked

Step 5: Enable BlogGo to Blog > Blog Setting. You should now get a message that the blog was installed. Fill in your preferred settings, and hit the Save button

The blog is now installed, and you can start by creating your first blog post.

The URL to the blog is by default: yourstore.com/index.php?route=blog/home



Additional information

- When you install the blog, a new layout is created called Blog (Design > Layouts)

- The blog extension comes with two custom modules for showing Blog categories and Latest blog posts. You find both these under Extensions > Modules in your admin panel.

Upgrading the blog extension

To upgrade the blog extension from version 1.X to 2.0, please follow the following steps:

1. Go to Blog > Blog settings, and save the content for Home title, Home page title, and Home description. This area is changed in 2.0, and you need to fill those fields again after upgrading.

2. Delete the file vqmod/xml/vqmod_blog.xml

3. Follow the default installation guide above. Replace your current files when uploading files in step 1. When you visit Blog > Blog Settings, you should see a success message about upgrade instead of about installation



Troubleshooting

After clicking the Refresh button, I get a error message like:
PHP Warning: fwrite() expects parameter 1 to be resource, boolean given in /../admin/controller/extension/modification.php on line 392

- This means the writing permissions is not correct. Please log in to your server, and set writing permission as 777 for the folder: system/modification
